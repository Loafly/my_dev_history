개발 포트폴리오를 관리할 수 있는 프로젝트

---

## 1. 소개 (Introduction)
- **프로젝트 배경**: 이 프로젝트는 개발 포트폴리오를 스스로 관리 하기 위해 생성
- **주요 기능**
    - 포트폴리오 프로젝트 / 경력 조회
- **작업 기간** : 2025.01.22 ~ 2025.02.25

## 2. 기술 스택 (Tech Stack)
- **Backend**: Spring Boot 3.1.4, JPA, QueryDSL, postgresql
- **Test**: JUnit 5, Mockito
- **etc**: Swagger
